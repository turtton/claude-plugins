# プランレビュー

ユーザが plan mode で作成した実装プランをレビューします。

## 手順

### 1. コンテキスト収集

- プロジェクトの CLAUDE.md があれば読み込む

### 2. エージェント起動

`plan-reviewer` エージェントを Task ツールで起動してください。

プラン内容はあなたのコンテキストに既にあります（自分が書いたため）。そのまま渡してください。

```
Task(
  subagent_type="plan-reviewer",
  description="プランレビュー",
  prompt="""
以下の実装プランをレビューしてください。

## レビュー対象のプラン
{プラン内容}

## プロジェクト設定（CLAUDE.md）
{CLAUDE.md の内容（あれば）}
"""
)
```

### 3. 結果の提示

- **指摘あり**: 指摘事項を表示し、プランの修正を促す
- **指摘なし**: その旨を伝える
